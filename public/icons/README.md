# PWA 图标说明

本目录需要以下图标文件（PNG 格式）：

| 文件名 | 尺寸 | 用途 |
| --- | --- | --- |
| `icon-192x192.png` | 192×192 | PWA 标准图标 |
| `icon-512x512.png` | 512×512 | PWA 大图标 / 启动画面 |
| `icon-maskable-512x512.png` | 512×512 | 带安全区的可遮罩图标 |
| `favicon.ico` | 32×32 | 浏览器标签图标 |
| `apple-touch-icon.png` | 180×180 | iOS 主屏图标 |
| `og-image.png` | 1200×630 | 社交分享卡片 |

## 设计要点

1. **主色**：`#FFB6C1`（樱粉）
2. **辅助色**：`#B8C5D6`（雾蓝）
3. **Logo 元素**：星星 + 麦克风的极简线条
4. **风格**：温柔、梦幻、女性向

## 临时占位

在正式图标生成前，可以使用 [PWA Builder Image Generator](https://www.pwabuilder.com/imageGenerator) 上传一张 512×512 的源图批量生成。
